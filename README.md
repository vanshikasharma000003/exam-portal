PROJECT TITLE:
University Exam Form Portal (Django)

DESCRIPTION:
This project is a web application developed using Django framework.
It allows students to log in, fill an exam form, and store the data in a database.

FEATURES:
1. User Login System (Authentication)
2. Dashboard after login
3. Exam Form submission
4. Data stored in SQLite database
5. Admin panel to view submitted data
6. View submitted forms page

TECHNOLOGIES USED:
- Python
- Django
- HTML
- Bootstrap (for UI)
- SQLite Database

PROJECT STRUCTURE:
- exam_portal (main project folder)
- portal (app folder)
- templates (HTML files)
- models.py (database structure)
- views.py (logic)
- urls.py (routing)

HOW TO RUN THE PROJECT:
1. Activate virtual environment:
   venv\Scripts\activate

2. Install dependencies:
   pip install django

3. Run server:
   python manage.py runserver

4. Open browser:
   http://127.0.0.1:8000/

LOGIN DETAILS:
Use superuser credentials created using:
python manage.py createsuperuser
